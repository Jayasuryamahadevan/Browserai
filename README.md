# Saturn Browser (Agentic AI OS Frontend)

**Saturn Browser** is a next-generation browser interface designed as a frontend for an Agentic AI Operating System. It integrates browsing, AI assistance, memory management, and project organization into a unified desktop experience.

## ‚ú® Features

- **AI-Powered Side Panel**: Direct integration with AI agents for assistance.
- **Memory & Context**: Persistent memory panel to track context across sessions.
- **Project Management**: Tools to organize tabs and resources into projects.
- **Modern UI**: Sleek, dark-themed interface built with Tailwind CSS.
- **Desktop Integration**: Full desktop capabilities via Electron.

## üöÄ Getting Started

### Prerequisites

- **Node.js**: [Download & Install](https://nodejs.org/) (Version 18+ recommended)
- **Git**: [Download & Install](https://git-scm.com/)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Jayasuryamahadevan/Browserai.git
   cd Browserai
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

### Running the App

You can start the application using the included helper script:

```bash
.\jessi.cmd
```

Or using standard npm commands:

```bash
npm run dev
```
